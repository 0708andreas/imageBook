Instrukser til Windows:
  • Download alle filerne ovenfor
  • Opret en genvej til Python på skrivebordet.
  • Gem zip filen på skrivebordet.
  • Højreklik på zipfilen og vælg "udpak alle"
  • Træk filen server_ui.py hen på genvejen til Python
  • Åbn din browser og gå ind på "http://localhost:8080"
  • Følg instrukserne på den side
Intrukser til Linux:
  • Download image_generator.zip
  • Åbn din terminal
  • Skriv følgende kommandoer i rækkefølge:
  • $ sudo apt-get install python-pip
  • $ sudo pip install PIL
  • Gå til kataloget med den downloadede zip-fil
  • $ unzip generator.zip
  • $ python server_ui.py
  • Åbn din browser og gå ind på "http://localhost:8080"
  • Følg instrukserne på den side
